# Spotify Data Analysis #

## This project is part of my learning journey in data analysis and visualization. I used a Spotify dataset to explore music trends and practice both data exploration and visualization. ##

### What I worked on
Performed Exploratory Data Analysis (EDA) using Python
Created basic KPIs and summary metrics
Built visualizations to explore trends in the data
Analyzed relationships between different audio features
Designed a simple dashboard layout in Tableau
### Tools used
Python (Pandas, Matplotlib / Seaborn)
Tableau
Excel / CSV dataset

### What I learned

I learned how to clean and explore data using Python and how to turn raw data into simple visual insights using dashboards.
